War Game Coursework for Software Development class
MSc in HPC with Data Science, University of Edinburgh. 

The code was developed and runs using Python2.7.

To run the updated program:
    $ python main.py

Required packages: json, random. 


Included files:
    main.py: Main function, performs gameplay. 
    
    game_functions.py: All functions required to edit data and start game.
    
    squad.json: Data file, containing all game actors and their stats. 
    
    roster.json: Data file, containing new members to buy.
